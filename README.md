# 🤖 AI Engineer Portfolio

A modern, production-ready portfolio website showcasing AI engineering projects and expertise. Built with cutting-edge web technologies for optimal performance, interactivity, and user experience.

## ✨ Features

- **Next.js 14**: Modern React framework with server-side rendering and static generation
- **3D Interactive Components**: Real-time 3D visualizations using Three.js and React Three Fiber
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **TypeScript**: Full type safety for robust development
- **SEO Optimized**: Meta tags and structured data for better search visibility
- **Performance Optimized**: Optimized images, code splitting, and lazy loading
- **Dark Mode Ready**: Beautiful theme system for enhanced user experience

## 🎯 Sections

- **Hero**: Eye-catching introduction with animated 3D elements
- **About**: Professional background and expertise overview
- **Experience**: Career timeline and professional achievements
- **Skills**: Technical skills organized by category
- **Projects**: Showcase of AI and ML projects with descriptions
- **Certifications**: Professional certifications and credentials
- **Contact**: Easy contact form and social links

## 🚀 Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/DevAzan786/AIEngineerPortfolio.git
cd AIEngineerPortfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Build & Deployment

### Production Build

```bash
npm run build
npm start
```

### Type Checking

```bash
npm run type-check
```

### Linting

```bash
npm run lint
```
